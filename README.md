# Online Examination System

This is a basic online examination system developed using PHP and MySQL.

## Features

- User registration and login
- Admin dashboard to manage exams
- Timer-based exam submissions
- Results tracking

## Setup

1. Clone this repository to your web server.
2. Import the SQL dump from the `database` folder into your MySQL database.
3. Configure the database connection in `sql.php`.

## Usage

1. Access the application through your web browser.
2. Students can register, log in, and take exams.
3. Admins can log in to create and manage exams.

## Technologies Used

- PHP
- MySQL
